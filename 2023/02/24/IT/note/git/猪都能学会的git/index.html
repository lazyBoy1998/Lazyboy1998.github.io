Git 简单来说是用于版本控制的，牛逼的地方在于它是分布式的，你问我什么是版本控制，问度娘，stackoverflow，博客园。本教程不负责理论，只负责如何教你懒人式操作

# 多人开发操作

多人开发很多操作都会有所限制，只有一个人开发的时候，git merge 是允许的，这不会产生开发冲突，多人场景下最好的方法是使用 git rebase，最大限度避开合并冲突
