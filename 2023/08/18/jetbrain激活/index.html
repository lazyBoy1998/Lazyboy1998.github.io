# 一、激活

激活的方式有很多种，这里用激活码的方式（Activation code）。

# 1、打开激活网址 ****https://jetbra.in/s

如无意外，网址将做一次跳转解析，并展示当前可供跳转的网址清单。

前面几个带地球的，状态为当前在线可用，点击其中任意一个在线可用的链接（如果有问题，可以换下一个试一下）。

# 2、下载激活包

下载上面的压缩文件，jetbra.zip，发布日期为 2022 年 8 月 1 日，解压到 IDEA 的同级目录下。

# 4、填写特定激活码

配置完，需使用特定的激活码，点击激活网站上 IDEA 栏的 Copy to clipboard，复制激活码到剪切板。

启动 IDEA，在 管理许可证 -> 激活 -> 激活码 处，粘贴 Ctrl + V，点击激活即可。

旧版本已安装官方汉化插件，并且自动存放在本地用户目录下。新版本启动时自动读取，项目列表和插件配置都不受影响。

# 二、写在最后

此时可以顺利使用 IDEA 了，这里补充几点说明：

  • 请支持购买正版，或者申请教育版、社区版。
  • 激活网址 https://jetbra.in/s，收藏后不易走失。
  • 激活文件可以解压到任意位置，配置为绝对路径即可，这里建议与 IDEA 放在一起。
  • 本方法同样适用于该家族的其他产品，复制激活码的时候选对应栏即可，激活文件可以共用。
