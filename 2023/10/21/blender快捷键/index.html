# 一、基本操作

前置小操作，笔记本电脑替换数字键盘，从导航栏找到编辑 -> 偏好设置

偏好设置 -> 输入 -> 模拟数字键盘

快捷键

键盘说明
ctrl +alt+Q进入四视图
0进入摄像机视角
1点模式
2边模式
3面模式
1（小键盘）正视图
3（小键盘）右视图
7（小键盘）后视图
/聚焦物体，将没有选中物体进行屏蔽
.
home 键盘
G移动
ALT+G移动物体归零
R旋转
ALT+R旋转归零
S缩放
ALT+S缩放归零
A全选物体
AA取消全选物体
L局部全选物体
H隐藏选中物体
E挤出面
alt+H反选
Shift+A物体添加（非编辑模式下起效）
Ctrl+ J合并
P分离
Shift+D复制
Shift+R重复上一次操作
Shift+ ~开启第一人称（wasd 键盘可移动）
ctrl+i反选
alt+E旋转

按 N 键 -> 锁定摄像机， 可以跟随视图操作

x 在 X 轴移动

Y 在 y 轴移动
