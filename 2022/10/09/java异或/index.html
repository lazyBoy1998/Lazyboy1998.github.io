# 异或介绍

异或是一种基于二进制的位运算，用符号 XOR 或者 ^ 表示，其运算法则是对运算符两侧数的每一个二进制位，同值取 0，异值取 1。

性质
1、交换律
2、结合律（即 (ab)c == a(bc)）
3、对于任何数 x，都有 xx=0，x0=x
4、自反性 A XOR B XOR B = A XOR 0 = A

# 异或运算的基本定理

  1. 异或满足结合律： (a^b)^c = a^(b^c)
  2. 异或满足交换律： a^b = b^a
  3. 任意数与自身异或得： a^a = 0
  4. 任意数异或 0 的到自身： a^0 = a
