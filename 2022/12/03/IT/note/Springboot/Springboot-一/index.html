# spring-boot-maven-plugin

ring-boot-maven-plugin 为 Spring Boot Maven 插件，提供了：

  1. 把项目打包成一个可执行的超级 JAR（uber-JAR）, 包括把应用程序的所有依赖打入 JAR 文件内，并为 JAR 添加一个描述文件，其中的内容能让你用 java -jar 来运行应用程序。

  2. 搜索 public static void main() 方法来标记为可运行类。
